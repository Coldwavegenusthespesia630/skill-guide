## 🚀 Running the software

Once you download the file and set up the utility, you can start the application.

1. Navigate to the folder where you saved the skill-guide file.
2. Double-click the file named skill-guide.exe.
3. A black window appears on your screen. This window keeps the program running.
4. Do not close this window while you use the application.
5. Open your web browser.
6. Type the address provided in the black window into your browser bar. Usually, this is http://localhost:3000.
7. Press Enter.

The presentation interface will load in your browser.

## 📋 Features

The application provides a system to track your AI agent skills.

- **Auto-Discovery:** The program finds your installed agent skills automatically. You do not need to hunt for folders or files.
- **HTML Slide Generation:** Each skill gets converted into an individual slide. You navigate these slides using your keyboard arrows.
- **Visual Clarity:** The layout focuses on readability. It removes technical noise so you see only what matters.
- **Workflow Support:** You track your agent progress through categories. This helps you understand which skills receive updates or changes.
- **Offline Access:** Once generated, the HTML files live on your computer. You do not need an internet connection to review your slides.

## 🛠️ Frequently asked questions

**Do I need an internet connection to use the tool?**
You only need the internet to download the file. The tool runs on your local machine without external servers.

**What happens if the black window closes?**
If the window closes, the application stops. Double-click the skill-guide.exe file again to restart the process.

**Can I change the slide design?**
The tool uses a standard theme. You can edit the underlying files if you understand basic web design, but the default setting works for most users.

**Does this software store my data online?**
No. All your data stays on your local hard drive. The application does not send information to cloud servers.

**How do I update the application?**
Visit the main GitHub page and download the newest release file. Replace the old file with the new one. 

## 🛡️ Troubleshooting

If you encounter issues, try these steps:

1. Check if Node.js is active. Open your Command Prompt and type `node -v`. If you see a version number, it works.
2. Ensure you downloaded the Windows version from the release page.
3. Check your firewall settings. Sometimes, Windows asks for permission to allow network access. Click Allow to let the application display the slides.
4. Clear your browser cache if the slides look outdated after you add new skills.
5. Restart your computer to clear any locked memory processes that might conflict with the software.
